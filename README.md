# Movie Rating Analysis

A Python-based Movie Rating Analysis web application built using Streamlit, Pandas, Matplotlib, and Seaborn.

## Features

* Analyze movie ratings
* Display top rated movies
* Rating distribution visualization
* Genre analysis
* Average rating calculation

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Streamlit

## Run the Project

```bash
py -m streamlit run app.py
```
